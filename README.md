# UI/UX Pro 設計風格庫 (UI/UX Pro Style Library)

這是一個由 [UI/UX Pro Max Skill](https://ui-ux-pro-max-skill.nextlevelbuilder.io/) 生成的綜合設計資源庫，包含了現代 UI/UX 風格展示、專業配色方案以及圖示庫。

## 專案概述 (Project Overview)

本專案旨在提供設計師與開發者一個快速參考的視覺化圖書館。主要包含三個核心模組：

1.  **風格庫 (Styles)**: `uiuxpro_styles.html`
    *   收錄 58 種現代 UI/UX 設計風格（如極簡主義、新擬物化、玻璃擬態、Bento Grid 等）。
    *   提供視覺特徵、適用場景、不適用場景、無障礙性分析及框架相容性資訊。
    *   支援分類篩選（如：一般、登陸頁面、商業智慧儀表板）。

2.  **配色庫 (Colors)**: `uiuxpro_colors.html`
    *   提供針對不同產品類型（SaaS, 電商, 金融, 醫療等）的專業配色組合。
    *   包含主要顏色、次要顏色、CTA、背景與文字色的具體 Hex 值。
    *   支援「隨機推薦」與「全部瀏覽」兩種模式。

3.  **圖示庫 (Icons)**: `uiuxpro_icons.html`
    *   基於 Lucide React 的精選圖示集合，展示常用圖示的最佳實踐。

## 技術棧 (Tech Stack)

*   **Frontend**: HTML5, CSS3 (CSS Variables, Flexbox, Grid), Vanilla JavaScript (ES6+)
*   **Data Processing**: Python 3 (用於將 CSV 數據轉換為 JSON)
*   **Data Source**: CSV (位於 `.shared/ui-ux-pro-max/data/`)
*   **Icons**: Lucide Icons (CDN)

## 使用方法 (Usage)

### 重新生成數據 (可選)
本專案的數據（`js/style-data.js` 和 `js/colors-data.js`）是由 Python 腳本從 CSV 檔案生成的。如果您修改了 `.shared/` 目錄下的 CSV 原始檔，請運行以下指令更新數據：

```bash
# 更新風格庫數據
python3 generate_data_js.py

# 更新配色庫數據
python3 generate_colors_js.py
```

### 功能亮點
*   **模組化設計**: CSS 與 JavaScript 邏輯已分離，易於維護。
*   **懶加載 (Lazy Loading)**: 風格庫中的預覽 iframe 採用懶加載技術，提升頁面載入效能。
*   **響應式設計**: 支援不同螢幕尺寸瀏覽。
